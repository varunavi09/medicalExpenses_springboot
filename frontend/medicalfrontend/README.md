# Medical Expense Manager 💊

## 📌 Description

A full-stack web application to manage and track medical expenses.
Users can add, update, delete, and view expenses with automatic calculation of total expense and remaining balance.

---

## 🚀 Tech Stack

* Frontend: React.js
* Backend: Spring Boot
* Database: MongoDB

---

## ✨ Features

* Add medical expense details
* Update existing records
* Delete records
* Automatic calculation of:

  * Total Expense
  * Remaining Balance
* Real-time data display

---

## ⚙️ How to Run

### 🔹 Backend (Spring Boot)

1. Open backend folder
2. Run the application

Runs on:
http://localhost:8080

---

### 🔹 Frontend (React)

1. Open frontend folder
2. Install dependencies

```
npm install
```

3. Start app

```
npm start
```

Runs on:
http://localhost:3000

---

## 🔗 API Endpoints

* GET `/expenses` → Fetch all expenses
* POST `/expenses` → Add new expense
* PUT `/expenses/{id}` → Update expense
* DELETE `/expenses/{id}` → Delete expense
